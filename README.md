IT-Term-Ref-Table
=================

English to Chinese Translation Table for IT Terminologies
IT術語中英文對照表


項目宗旨
=========

給出IT界英文術語的各種中文譯法，討論其優劣，並提出本社區推薦的譯法。


術語範圍
========

包括但不限於軟件設計領域、電子遊戲領域。

========
